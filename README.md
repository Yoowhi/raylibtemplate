# Raylib Template

A lightweight [raylib](https://www.raylib.com) project template for macOS, Linux, and Windows with VSCode task and launch configurations.

Click **Use this template** button in the top right corner to create your own repository.

Clone the project, then create `build` and `src/resources` directories.
